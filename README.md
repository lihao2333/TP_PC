# 北京邮电大学简易实验报告模板

A LaTeX template of report in required format in BUPT(most cases).

## Usage

Just follow the template and compile.

Only tested on TexLive 2017, Ubuntu 16.04, xelatex.

```bash
latexmk --xelatex report.tex
```

OR

```bash
xelatex report.tex
```
